# Conditioning 
>[!Definition] Conditioning
>$\text{Let A be an event in the same experiment as r.v. X}$
>$\text{Suppose, P(A)>0, we define the conditional pmf of X given A as}$
>$$P_{X|A}(x) = P(X=x|A)=\frac{{P(\{X=x\}\cap A)}}{P(A)}$$

>[!Example] Conditioning example
>$\text{Bag with 3 coins, 2 fair, and with 2 heads}$
>$\text{Let } X=\begin{cases}1, \text{ if a toss is head} \\0, \text{ otherwise}\end{cases}~~~~~,~~~ A = \{\text{we drew a fair coin}\}$
>$P_{X}(x) = \begin{cases}{\frac{2}{3} \frac{1}{2} + \frac{1}{3}} ~~~~~x = 1\\{\frac{2}{3} \frac{1}{2}}~~~~~~~~~~~~~x = 0\end{cases}$
>$P_{X|A}(x) = \begin{cases}{\frac{1}{2}}~~,~x=1\\{\frac{1}{2}}~~,~x=0\end{cases}$

>[!Example] Dice Roll Conditional
>$\text{Let X be a roll of a fair 6-sided die}$
>$A = \{\text{roll is even}\}$
>$P_{X}(x) = \frac{1}{6}~~,~~x=1,2,3,\dots,6$
>$P_{X|A}(x) = P(X=x|A)=\begin{cases}{\frac{P(X=2)}{P(A)}} = \frac{1}{3}~~~~~x=2\\{\frac{P(X=4)}{P(A)}} = \frac{1}{3}~~~~~x=4\\{\frac{P(X=6)}{P(A)}} = \frac{1}{3}~~~~~x=6\end{cases}$

>[!Definition] Conditioning of Multiple R.V.S
>$\text{Let X and Y be two r.v.'s associated to the same experiment}$
>$\text{The conditional pmf of X given }\{Y=y\} \text{ is given by}$
>$$P_{X|Y}(x|y) = P(X=x|Y=y) = \frac{{P(\{X=x\}\cap\{Y=y\})}}{P(Y=y)}=\frac{{P_{X,Y}(x,y)}}{P_{Y}(y)}~~,$$
>$$P_{Y}(y)>0$$

>[!Example] Children using Conditioning
>$\text{The number of children in a family is modeled as follows}$
>$$N = \# \text{ of children},~~~P_{N}(n)=\begin{cases}{\frac{1}{10}}~~,~~n=0\\{\frac{3}{10}}~~,~~n=1\\{\frac{4}{10}}~~,~~n=2\\{\frac{2}{10}}~~,~~n=3\end{cases}$$
>$\text{Each child born to a family is equally likely to be a boy or a girl}$
>$\text{Let Y be the \# of boys in a family}$
>$GOAL: \text{Pmf of Y}$
>$P_{Y}(y)~~~~,y\in\{0,1,2,3\}$
>$P_{Y|N}(y|n)~=~\binom ny\left( \frac{1}{2} \right)^{y}\left( \frac{1}{2} \right)^{n-y},~~~y=0,\dots,n$
>$Y|N=n ~~Binomial\left( n,{\frac{1}{2}} \right)$
>$N=0,~~~\text{no children}\implies \text{no boys}$
>$$P_{Y|N}(y|0) = 1,~~\text{if y=0}$$
>$N=1$
>$$P_{Y|N}(y|1) =\begin{cases}{\frac{1}{2}},~~y=0\\{\frac{1}{2}},~~y=1\end{cases}$$
>$N=2$
>$$P_{Y|N}(y|2) =\begin{cases}{\binom 20(\frac{1}{2}})^{2},~~y=0\\{\binom 21(\frac{1}{2}})^{2},~~y=1\\{\binom 23(\frac{1}{2}})^{2},~~y=2\end{cases}$$
>$N=3$
>$$P_{Y|N}(y|2) =\begin{cases}{\binom 30(\frac{1}{2}})^{3},~~y=0\\{\binom 31(\frac{1}{2}})^{3},~~y=1\\{\binom 32(\frac{1}{2}})^{3},~~y=2\\{\binom 33(\frac{1}{2}})^{3},~~y=3\end{cases}$$
>$$P_{Y}(y) = \sum_{n=0}^{3}P_{N}(n)P_{Y|N}(y|n)$$
>$$=\sum_{n=y}^{3}P_{N}(n)\binom ny\left( \frac{1}{2} \right)^{n}$$

# Conditional Expectation and Variance
>[!Definition] Conditional Expectation
>$\text{Let X and Y be two r,v's associated to the same experiment}$
>$\text{Let A be an event with }P(A)>0, \text{ then}$
>$\text{The conditional expectation of X given A is,}$
>$$E[X|A] = \sum_{x}xP_{X|A}(x)$$
>$\text{and if g is any function}$
>$$E[g(x)|A]=\sum_{x}g(x)P_{X|A}(x)$$

>[!Theorem] Total Expectation
>$$E[X] = \sum_{y}E[X|Y=y]P_{Y}(y) ~~~\text{for any r.v.'s X and Y}$$
